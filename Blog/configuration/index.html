Blog 配置

发表于 2022-04-27 更新于 2023-04-05 分类于 Blog 本文字数： 956 阅读时长 ≈ 1 分钟

本文主要讲述Hexo + Next的基本Blog主题配置方法。

配置教程

目前已完成的主要是一些实用简单的主题配置。在主题配置时，参考了一些链接，列举如下：

  1. Next 官方blog 主题配置文件

    主要参考关于边栏头像的设置

  2. CC协议设置，将会在边栏和每篇文章的底部注明文章将采用 Creative Commons 协议 进行共享。最初了解到这个协议是源自一部《互联网之子》的纪录片，不过这就是另外的故事了……

  3. 最开始的时候也改变了自己配置主题的方式，就是在根目录下面重新创建了一个_config.next.yml文件，用于更加模块化管理自己界面，这个方法来自官方说明的主题配置方式，同时 Next主题 也给出了具体的使用方法

  4. 头像(avatar)的设计主要采用了 Figma 在线平台。由于头像是采用像素点阵进行设计的，导出为.svg格式会出现栅格化的现象，观感不佳。目前采用的是.png格式图像，尽管损失了一点图像质量，但是弥补了最大的缺陷
